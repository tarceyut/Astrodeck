# Astrodeck
AstroDeck is a C-based astronomy observation management system developed as the final project for the Programming Under Open Platforms course at the Universidad de Costa Rica.

Inspired by cyberdeck-style portable computing systems, the project aims to become the software core of a future portable astronomy observation platform built around a Raspberry Pi and custom hardware.

## Features
- Register personal observations.
- View registered observations.
- Search observations by object name.
- Linked list implementation.
- Dynamic memory management.

## Planned features
- File handling for saving and loading observations.
- Command-line arguments.
- Improved input validation.
- Future integration with embedded or Linux-based platforms.

## Technologies
- C.
- Linux.
- GCC.
- Dynamic Memory Allocation.
- Linked Lists.
